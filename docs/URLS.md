# LumenLink URLs

Canonical URLs for LumenLink services.

## Production

| Service | URL | Purpose |
|---------|-----|---------|
| **Website** | https://lumenlink.org | Main website, documentation, downloads |
| **API** | https://api.lumenlink.org | Backend Rendezvous API (config, attestation, gateways) |

## Development / Testing

| Service | URL | Purpose |
|---------|-----|---------|
| **Website** | http://localhost:3000 | Local development only |
| **API** | http://localhost:8080 | Local development only |

**Note:** localhost is for development and testing only. Production deployments must use [lumenlink.org](https://lumenlink.org) and [api.lumenlink.org](https://api.lumenlink.org).
